#Title: Document Categorization Using Machine Learning.

## Description
This project implements a Machine Learning-based Document Categorization System that automatically classifies textual documents into predefined categories. The system is designed to handle large volumes of unstructured data and organize them efficiently using intelligent algorithms.

## Prerequisites
Before running the project, make sure the following software is installed on your system:

- Python 3.14.0
- pip (Python package manager)
- Virtual Environment (recommended)
- Git
- Visual Studio Code or any code editor
- Required Python libraries:
  - Django
  - scikit-learn
  - pandas
  - numpy
  - joblib

## Installation Steps

1. Clone the repository:
   https://github.com/secure-web2025/projects-2526-TY.git

2. Navigate to the project folder:
   cd Group17_Document_Categorization

3. Create a virtual environment:
   python -m venv venv

4. Activate the virtual environment:

   For Windows:
   venv\Scripts\activate

5. Install required dependencies:
   pip install -r requirements.txt

6. Apply database migrations:
   python manage.py migrate

7. Start the Django development server:
   python manage.py runserver

8. Open the browser and visit:
   http://127.0.0.1:8000/

9. Upload or enter document text and get categorization results.
