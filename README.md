# Water Jar Management System

## Project Description

The Water Jar Management System is a web-based application developed using React and MongoDB to manage water jar distribution efficiently. It helps in handling customer records, tracking deliveries, and managing orders in an organized way.

## Software Installation Instructions

### 1. Objective

To install and configure all necessary software and tools required for developing and running the Web Application for Water Jar Distribution System (React + MongoDB).

---

### 2. Requirements

* Computer / Laptop
* Internet Connection
* Node.js installed
* MongoDB Compass installed
* Code Editor (VS Code recommended)
* Project ZIP file

---

### 3. Procedure / Steps

1. Unzip the project folder to a desired location on your computer.

2. Install Node.js from the official website if not already installed.

3. Open the **frontend folder** in terminal.

4. Run:

   ```
   npm install
   ```

   to install dependencies.

5. Start frontend:

   ```
   npm start
   ```

6. Ensure MongoDB is installed and running OR connect using MongoDB Compass.

7. Configure database connection if required (check `.env` file).

8. Open the **backend folder** and run:

   ```
   npm install
   ```

9. Start backend server:

   ```
   node app.js
   ```

   or

   ```
   nodemon app.js
   ```

10. Open browser and go to:

```
http://localhost:3000
```

---

### 4. Expected Output

The React application will run successfully in the browser and connect to the MongoDB database.
